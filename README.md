# w21-survey

Web App Deployment: <https://w21-survey.web.app/>

Firebase Database: <https://console.firebase.google.com/project/w21-survey/>

# Firebase Commands Reference

```bash
# install firebase
sudo npm install -g firebase-tools
# login
firebase login
# start local host
firebase serve
# publish
firebase deploy
```
